# lotto666

A simple lotto game on blockchain, selecting 6 numbers from 66

## start

yarn install

yarn run dev

## Contract

yarn hardhat compile
yarn hardhat test

edit .env

then:

yarn hardhat run scripts/deployFuji.js --network fuji
